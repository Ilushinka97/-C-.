# -C-.
Лабораторная работа по дискретной математике

Необходимо написать программное приложение на языке С, осуществляющее получение по таблице истинности заданной в соответствии с вариантом логической функции   СДНФ и СКНФ.
При этом требуется:
- таблица истинности должна загружаться в память ЭВМ из файла.
- заданная логическая функция должна выводиться на экран в виде карты Карно.

Теорема о существовании СДНФ:
Всякая булева функция (кроме 0) имеет единственную СДНФ.

Теорема о существовании СКНФ:
Всякая булева функция кроме 1 имеет единственную СКНФ.

lb2.cpp - основное тело программы.
Dynamic.cpp - фаил с пользовательской функцией, описывающей динамическое выделение памяти для двумерного массива.
GrayKarno.cpp - фаил с пользовательскими функциями, описывающими построение кода Грея и карты Карно.
openinput.cpp - фаил с пользовательскими функциями, описывающими открытие файла ".txt", считывание матрицы, находящейся в нём, и вывод её на консоль.  
postrSNF.cpp - фаил с пользовательскими функциями, описывающими построение СДНФ и СКНФ.
verif.cpp - фаил с пользовательской функцией, описывающим проверку на наличие символов, отличных от 0 и 1 проверка на пустое отношение или на отсутсвие матрицы.
